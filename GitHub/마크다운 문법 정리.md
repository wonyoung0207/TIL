## 텍스트 문법

_기울임_ 은 " _ text _ "로 감싸면 된다.

**굴은선** 은 "** text ** "로 하면 된다.

~~취소선~~ 은 " ~~ text ~~ "로 감싸면 된다.

really **_very_** import text...

this text is **_really important_**

## 텍스트

텍스트 한줄 코드로 변경하고 싶으면 " ` " 로 감싸준다.

`printf("hello world")`

코드 여러줄 변경하려면 " ``` "를 넣으면 코드블록이 생긴다.

코드블록은 사용할 언어를 선택할 수 있다.

```python
for i in range(5):
	print(i);
```
