# 자율주행 표준메시지

---

>



| **메시지 명칭** | **풀네임**              | **핵심 의미 및 역할**                                        | **비고**         |
| --------------- | ----------------------- | ------------------------------------------------------------ | ---------------- |
| **BSM**         | Basic Safety Message    | **차량의 상태 정보**. 위치, 속도, 방향, 브레이크 상태 등을 주변에 방송(V2V 중심). | 가장 기본        |
| **PVD**         | **Probe Vehicle Data**  | **차량 주행 이력**. 차량이 지나온 경로, 연비, 상태 정보 등을 관제 서버로 전송(V2I). | **관제 핵심**    |
| **SPaT**        | Signal Phase and Timing | **신호등 정보**. 현재 신호가 무엇인지, 다음 신호까지 몇 초 남았는지 전송. | 교차로 자율주행  |
| **MAP**         | Map Data                | **도로 정밀 지도**. 교차로의 차선 정보, 정지선 위치 등 복잡한 도로 형상을 설명. | SPaT와 짝꿍      |
| **TIM**         | Traveler Information    | **도로 상황 안내**. 공사 구간, 결빙, 사고 정보 등 운전자가 알아야 할 정보. | 도로 전광판 역할 |
| **RSA**         | Roadside Alert          | **위험 경고**. "앞에 구급차 이동 중", "기차 통과 중" 같은 즉각적 위험 알림. | -                |

## 핵심 데이터 형태 

| **구분**            | **용어** | **풀네임**              | **역할 및 데이터 특징**                                     | **웹 개발 비유**                           |
| ------------------- | -------- | ----------------------- | ----------------------------------------------------------- | ------------------------------------------ |
| **메시지** (Data)   | **PVD**  | Probe Vehicle Data      | 차량의 주행 이력(경로, 속도 등)을 관제소로 보내는 데이터    | **사용자 활동 로그** (Google Analytics 등) |
| **메시지** (Data)   | **RSA**  | Roadside Alert          | 도로 위 위험 상황(사고, 장애물)을 차량에 알리는 긴급 메시지 | **Push 알림 / Alert**                      |
| **장치** (Hardware) | **RSU**  | Road Side Unit          | 도로변에 설치되어 차량과 데이터를 주고받는 기지국           | **엣지 게이트웨이 / 공유기**               |
| **장치** (Hardware) | **ACU**  | Autonomous Control Unit | 차량 내부에 탑재되어 자율주행 알고리즘을 실행하는 두뇌      | **메인 서버 / 로컬 워커**                  |
| **메시지** (Data)   | **ACC**  | Adaptive Cruise Control | 첨단 운전자 보조 시스템 에서 가장 핵심적인 기능             | 차량 로그                                  |
