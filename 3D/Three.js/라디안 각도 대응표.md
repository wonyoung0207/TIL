## 라디안 ↔ 도(degree) 대응표

| 도(deg) | 라디안          |
| ------- | --------------- |
| 0°      | 0               |
| 1       | `Math.PI / 180` |
| 10      | `Math.PI / 18`  |
| 15      | `Math.PI / 12`  |
| 22.5    | `Math.PI / 8`   |
| 30°     | `Math.PI / 6`   |
| 45°     | `Math.PI / 4`   |
| 60°     | `Math.PI / 3`   |
| 90°     | `Math.PI / 2`   |
| 180°    | `Math.PI`       |
| 360°    | `Math.PI * 2`   |
