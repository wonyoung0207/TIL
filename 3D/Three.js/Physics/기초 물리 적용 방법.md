# 기초 물리 적용 방법

---

>

## 방법

1. 물리 world 생성 

   ```js
   this.world = new RAPIER.World({ x: 0.0, y: -9.81, z: 0.0 })
   ```

2. 물리 world의 scene 에 mesh 추가 

   - **물리 세계(Physics World)**랑 **렌더링 세계(Visual World)**를 *아예 분리해놓고*,
      `entity` 객체 안에서 둘을 **느슨하게 연결해서 같이 관리**하는 방식

   ```js
   const visual = new THREE.Mesh(
           new THREE.BoxGeometry(1 * 2, 0.5 * 2, 1.5 * 2),
           new THREE.MeshNormalMaterial({ wireframe: true })
       )
   this.game.world.scene.add(visual)
   
   this.chassis = this.game.physics.addEntity(
               physical
               visual
           )
   ```
