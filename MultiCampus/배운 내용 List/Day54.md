## NCP (Naver Cloud Platform)

### Cloud 환경 

 	1. 인프라 ( IAAS )
     - 리눅스를 이용 
     - JDK, Mysql , Web Server 를 설치해서 진행 
 	2. 플렛폼 ( PAAS )
     - 네이버에서 제공하는 AI 플렛폼 이용 
 	3. 소프트웨어 ( SAAS )
