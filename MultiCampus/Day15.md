> Frame 객체를 이용해 UI를 개발해본다. 

## 1. **Frame**
   + java.awt 에 있는 기능으로, 사용자 UI 구성에 사용된다. 
   + 기능 구현
     1. inint()
        + 컴포넌트들을 **생성**하는 곳
     2. make()
        + frame의 **외형**을 변경하는 곳 
        + 컴포넌트 추가 순서도 중요하다. 
     3. addEvent()
        + frame에 부여할 **이벤트를 추가**하는 곳
