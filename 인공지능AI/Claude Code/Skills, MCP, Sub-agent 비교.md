## **탄생 타임라인**

- **2024년 11월:** MCP (Model Context Protocol) 발표 - Anthropic이 외부 시스템 연결 표준 제시
- **2025년 7월 25일:** Sub-agents 출시 - Claude Code에서 전문화된 AI 어시스턴트 등장
- **2025년 10월 16일:** Skills 발표 - 재사용 가능한 워크플로우 패키지 시스템

Sub-agents가 먼저 나왔고, Skills가 약 3개월 후에 등장했다. Skills 등장 전까지는 Sub-agents가 비슷한 역할을 했다.

## 사용시점

**Skills만으로 충분한 경우:**

- 문서 생성 (DOCX, PDF, PPTX, XLSX)
- 반복적인 워크플로우 자동화
- 표준화된 작업 (특정 문서 모델링, 프레젠테이션)

**MCP가 필요한 경우:**

- 외부 데이터베이스 연결 (BigQuery, PostgreSQL)
- API 통합 (Slack, GitHub, Jira)
- 파일 시스템 접근

**Sub-agents가 유용한 경우:**

- 복잡한 작업을 병렬로 처리
- 특정 전문 분야 (보안, 성능 최적화, 데이터 분석)
- 메인 컨텍스트 절약 필요
