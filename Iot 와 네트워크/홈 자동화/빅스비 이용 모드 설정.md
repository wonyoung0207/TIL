## 방법1 ( 가상 스위치 이용)  * 가장 추천!! * 

- 스마트 싱스 앱 메뉴 -> 실험실 -> 가상스위치에서 +(플러스)눌러서 가상스위치를 만들기
- 빅스비 음성 명령으로 가상 스위치 호출 
  - 나는 스위치 이름을 "00 모드" 라고 설정 -> 빅스비 단축 명령 -> "00 모드" 호출시 "00 모드 켜줘" 로 음성 인식 추가 
  - 단, 이런식으로 했을 경우 해당 폰에서만 음성 인식이 되므로 멤버가 동일하게 사용하기 위해선 빅스비 단축 명령 등록을 해줘야한다. 
  - 단축 명령 하기 싫다면 "00 모드 켜줘" 라고 요청해 가상 스위치 on/off 를 해야한다. 


## 방법2 ( 빅스비 단축 명령어 이용)

- 설정에서 빅스비 설정 찾고, 오른쪽 상단 점세개 버튼 누르면 "단축 명령어" 등록할 수 있다. 
- 갤럭시 홈미니 스피커 설정에도 단축명령어가 있다. 

## 방법3 ( 루틴으로 만들어 처리)

- 세가지 액션을 하나의 수동루틴으로 만드시고 음성명령에 해당 수동루틴을 실행하도록 하시면 됩니다.
