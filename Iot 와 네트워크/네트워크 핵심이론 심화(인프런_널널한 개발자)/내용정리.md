## 세가지 네트워크 장치 구조 

1. Inline
   1. 라우터
   2. 차단, 허용, 필터링 
2. Out Of Path
   1. 센서
   2. 패킷 
3. Proxy
   1. 소켓 스트림 수준의 File 
   2. 패킷을 조립하기엔 내용이 클 경우 사용 

## Inline 구조

1. 종류
   1. 공유기, Router, IPS 
2. 데이터 단위 
   1. Packet 
3. 특징 
   1. 네트워크 속도와 많은 연관이 있다. 

## Out Of Path

1. 종류
   1. 센서 + Packet 수집장치 
   2. Tab 스위치
