# [ IEEE 종류 ]

## < 802.1 >

- 브릿지 및 LAN 규격을 다루는 표준 

## < 802.2 >

- 논리적 링크 제어 서비스 제공
- 데이터 링크(L2) 에서 작동하는 프로토콜을 규정 
- LLC

## < 802.3 >

- 이더넷 표준 정의 
- L1과 L2에서 동작
- CSMA/CD
- 유선 네트워크 

## < 802.4 >

- Token Bus 

## < 802.5 >

- Token Ring 

## < 802.6 >

## < 802.7 >

## < 802.8 >

- Fiber Optic Lan 

## < 802.11 > 

- Wireless Network 
- 무선 네트워크 (WLAN) 표준

- WPAN표준
