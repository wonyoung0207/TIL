# 모터(풍량) vs 댐퍼(차단) 비교

- 모터
  - 엔진과 같음. 연기를 밖으로 버리는 능력 
- 전동댐퍼
  - 체크밸브와 같음. 다른 집 냄새가 우리 집으로 못 오게 막음.
- 시너지
  - 후드 ON → 댐퍼 열림 + 모터 작동 / 후드 OFF → 댐퍼 닫힘 (밀봉).
- 결론: 
  - 흡입력이 좋아도 댐퍼가 없으면 남의 집 냄새를 맡아야 함

##### 모터 흡입력 vs 전동댐퍼 차이점

| **구분**      | **후드 모터 (Motor)**                       | **전동댐퍼 (Motorized Damper)**             |
| ------------- | ------------------------------------------- | ------------------------------------------- |
| **핵심 역할** | 주방의 오염된 공기를 밖으로 **밀어내는 힘** | 밖에서 들어오는 냄새, 벌레, 소음을 **차단** |
| **성능 지표** | 풍량                                        | 기밀성 (밀폐력)                             |
| **작동 원리** | 전기가 들어오면 팬이 회전함                 | 후드를 켜면 문이 열리고, 끄면 꽉 닫힘       |
| **필요성**    | 음식을 할 때 반드시 필요함                  | **공동 주택(아파트)**에서 필수적임          |

## 후드 모터와 댐퍼 전기 연결

##### 방법

1. **동작 원리:** 사용자가 후드 스위치를 켬 → 모터에 220V 전류가 흐름 → 동시에 연결된 전동댐퍼에도 220V가 흐름 → 댐퍼 모터가 구동되어 문이 열림.
2. **복귀 원리:** 후드를 끔 → 전류 차단 → 전동댐퍼 내부의 **복귀 스프링**이나 **잔류 전력**을 이용해 문이 꽉 닫힘.

##### 확인할 점 

- **전압 확인:** 
  - 대부분의 주방 후드와 전동댐퍼는 **AC 220V**를 사용하지만, 간혹 DC 제품이 섞여 있을 수 있어 반드시 전압이 일치하는지 확인필요
- **풍량 조절(단수) 문제:** 
  - 일반적인 슬라이딩 후드는 1단, 2단 스위치가 따로 있다.
  - 이때 **공통선(Common)**이 아닌 특정 단수 선에만 연결하면, 1단일 때는 열리는데 2단일 때는 닫히는 황당한 상황이 생길 수 있어 반드시 모든 단수에서 전기가 흐르는 '공통 출력선'을 찾아야 함.
- **결속 마감:** 
  - 후드 배관 안은 유증기(기름기)가 많습니다. 
  - 전선 연결 부위가 헐거우면 화재 위험이 있으므로, 반드시 **와고(Wago) 커넥터**를 쓰거나 절연 테이프로 꼼꼼히 마감해야 합니다.
